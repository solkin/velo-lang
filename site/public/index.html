Velo by Example

Simple. Embeddable. Yours.

Velo Lang is a functional, strictly-typed, compilable programming language that runs on a lightweight stack-based virtual machine. It features strict typing, higher-order functions, generics, and easy embedding into JVM applications.

Velo by Example is a hands-on introduction to Velo using annotated example programs. Check out the first example or browse the full list below.
