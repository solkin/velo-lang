# Language Basics

## Hello, World!

```velo
include "lang/terminal.vel";

str hello = "Hello, World!";
term.println(hello);
```

## Comments

Velo Lang uses single-line comments starting with the `#` symbol:

```velo
# This is a comment
str text = "Hello"; # Comment at end of line
```
