# Variables and Assignment

## Variable Declaration

```velo
# With explicit type
int x = 10;
str name = "Velo";

# With any type
any value = 42;
value = "Hello";
```

## Assignment

```velo
int a = 5;
a = 10;              # Change value
a = a + 1;           # Arithmetic operations
```
